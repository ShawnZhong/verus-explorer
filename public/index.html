verus-explorer

rustc → VIR → AIR → Z3 (vendored, wasm32)

Rustc front-end invoked directly through rustc_interface::run_compiler (skipping rustc_driver's CLI/ICE/panic-hook glue), source fed via Input::Str. Extern-crate resolution is served out of a prebuilt wasm-libs bundle (core, alloc, compiler_builtins, vstd, verus_builtin, and the Verus macro-stub rmetas) fetched from ./wasm-libs/; rustc sees them via a virtual --sysroot=/virtual wired through rustc_session::filesearch::sysroot::Callbacks. #![no_std] is injected via -Zcrate-attr to keep the std import out (only core is bundled). After HIR lowering, Verus takes over: HIR → simplified VIR → ast_to_sst → poly → AIR → Z3, with each stage exposed via the toggles below.
